# HomeTrap — приведення стилів до затвердженого макета

## Overview

Візуальне рев'ю живого порталу (localhost:5173, усі екрани, light + dark) проти
затвердженого макета показало: функціонал повний, але стилі помітно розійшлися.
Мета циклу — привести UI до макетних дизайн-токенів і виправити помічені
косметично-форматні дефекти. Зміни **суто frontend** — жодних змін API чи БД.

Знахідки рев'ю:
1. **Шапка**: двоповерхова (лого-рядок + окремий ряд навігації) замість одного рядка;
   активна вкладка — бліда плашка з темним текстом замість суцільної шавлієвої з білим;
   чип курсу «USD · 44.75 ₴» (крапка) замість «USD НБУ 44,75 ₴»; лого — текст «HT»
   замість іконки будиночка.
2. **Картки/плитки**: без hairline-бордерів, тіні сильніші за макетні; eyebrow плиток
   без uppercase/letter-spacing; немає кольорових приміток (позитив/негатив).
3. **Формати чисел**: тарифи «7.95689 ₴», «197.91000 ₴» (хвости нулів, крапка замість
   коми); показники «9582.000» замість «9 582»; курс у полі «44,791700».
4. **Дашборд**: рядки квартир без аватара-сквіркла з літерою; назва дублюється адресою;
   «Потребує уваги» без кольорових крапок-маркерів пріоритету.
5. **Картка квартири**: вузька картка «Реквізити» замість рядка фактів на всю ширину;
   нативний file input «Choose Files»; телефон орендаря — дефолтний синій лінк;
   таблиця послуг без кольорових крапок-маркерів (газ/світло/вода).
6. **Статистика**: кольори графіків приглушені (лінії ~#c88761/#d5aa3f/#6798ad, бари
   сейдж+беж — майже не розрізняються) замість валідованої палітри макета; немає
   заливки-області під лініями споживання; **бар від'ємного місяця не рендериться
   взагалі** (вересень 2025: коригування −10 740 ₴ → порожній слот і «висячий» підпис
   під віссю).

## Development Approach

- **testing approach**: Regular (спочатку код, потім тести в межах того ж таска)
- завершувати кожен таск повністю перед переходом до наступного
- малі сфокусовані зміни; функціональна поведінка не змінюється
- **CRITICAL: кожен таск МУСИТЬ включати нові/оновлені тести** (success + error)
- **CRITICAL: усі тести мають проходити перед початком наступного таска**
- **CRITICAL: оновлювати цей план при зміні скоупу під час реалізації**

## Solution Overview

Джерело правди — дизайн-токени макета в `theme.css` (light + dark). Компоненти
підключаються до токенів; хардкод кольорів у `Stats.tsx` (`seriesColor()`) замінюється
на CSS-змінні палітри графіків. Числові формати централізуються в `utils/format.ts`
(нові `formatTariff`, `formatReading`, `formatRate`) і застосовуються всюди, де зараз
сирі значення API. Розмітка шапки, плиток, рядків квартир і фактів картки квартири
приводиться до макетної структури.

## Technical Details

**ВАЖЛИВО — стратегія токенів:** у кодовій базі вже є токени з іменами `--color-*`
(`--color-bg`, `--color-surface`, `--color-text`, `--color-primary`, `--color-border`
тощо), на які зав'язані `Layout.css` і `portal.css`. Імена НЕ перейменовуємо —
оновлюємо **значення** наявних токенів до макетної палітри і лише **додаємо** нові
(`--chart-*`, `--color-on-primary`). `theme.test.ts` пишеться проти реальних імен.

Мапінг макет → наявний токен (light / dark):

| Наявний токен | Значення макета light | dark |
|---|---|---|
| `--color-bg` | `#F6F5F1` | `#141613` |
| `--color-surface` | `#FFFFFF` | `#1F211D` |
| `--color-surface-muted` | `#FBFAF7` | `#24261F` |
| `--color-text` | `#2F3833` | `#E9EAE4` |
| `--color-text-muted` | `#8C958E` | `#8A9089` |
| `--color-border` | `#E7E5DD` | `#33362F` |
| `--color-primary` (зараз `#71836b`) | `#4E7A6A` | `#7FAB97` |
| `--color-primary-soft` | `#DFEAE4` | `#2A3630` |
| `--color-on-primary` (ДОДАТИ) | `#FFFFFF` | `#14251E` |
| `--chart-gas` (ДОДАТИ) | `#eb6834` | `#d95926` |
| `--chart-elec` (ДОДАТИ) | `#eda100` | `#c98500` |
| `--chart-water` (ДОДАТИ) | `#2a78d6` | `#3987e5` |
| `--chart-rent` (ДОДАТИ) | `#1baf7a` | `#199e70` |
| `--chart-util` (ДОДАТИ) | `#4a3aa7` | `#9085e9` |

Картки: radius 12px (звірити, який із наявних `--radius-sm/md/lg` використовують
картки, і виставити йому 12px), бордер 1px `--color-border`, `--shadow-card` light
`0 1px 2px rgba(47,56,51,.05), 0 4px 14px rgba(47,56,51,.04)`; у dark-темі
`--shadow-card: none` явно (це спільний композит — правити його, а не лише колір тіні).
Eyebrow: 11px, uppercase, letter-spacing .09em, `--color-text-muted`, 600.
Статусні плашки (tint bg + tint ink) — уже реалізовані, звірити відтінки.

Точність курсу: чип у шапці та плитка курсу на дашборді — фіксовано 2 знаки
(«44,75 ₴»); поле курсу в редакторі рахунку — 2–4 значущі знаки з тримом нулів
(«44,7917»).

Формати чисел (uk-UA, кома як десятковий розділювач):
- `formatTariff("197.91000")` → «197,91»; `("7.95689")` → «7,95689» (обрізати лише
  хвостові нулі, мінімум 2 знаки після коми)
- `formatReading("9582.000")` → «9 582» (цілі, розділювач тисяч; дробова частина
  відкидається лише якщо нульова, інакше показується без хвостових нулів)
- `formatRate("44.791700")` → «44,7917» (2–4 значущі знаки після коми, трим нулів)

Від'ємний місяць у стековому графіку: бар не малюється (стек від нуля некоректний),
натомість у слоті на базовій лінії — маркер-ромб «коригування» з тултіпом, що показує
реальні значення (оренда, комунальні з мінусом, разом); підпис суми — у тултіпі, не
під віссю.
